# Nexus Atom Store

Это каталог готовых активов (скиллы, шаблоны, профили).

⚠️ **Важно:** В продакшене этот каталог живёт на удалённом сервере/репозитории.
В данном репозитории он смоделирован локально для разработки и тестирования.

## Структура
```
store/
├── skills/       # Готовые скиллы для агентов
├── templates/    # Шаблоны артефактов
└── profiles/     # Готовые профили инициализации (minimal, standard, enterprise)
```

## Версионирование
Все активы используют SemVer: `skills/notebook/1.0.0/`
Папка версии неизменяема (immutable).
