# Workflow: Быстрый захват идеи

## Триггер
Пользователь говорит: "Запиши: [текст идеи]" или "У меня мысль: [текст]".

## Шаги
1. **Извлечь суть**: Выделить заголовок и основной текст из сообщения.
2. **Сгенерировать slug**: Транслитерация заголовка для имени файла.
3. **Создать файл**: 
   - Путь: `workspace/inbox/2026-04-13_{slug}.md`
   - Frontmatter: `status: raw`, `tags: [idea]`
4. **Подтвердить**: Сообщить пользователю: "✅ Идея записана. Хочешь добавить теги?"
5. **Опционально**: Если пользователь просит теги — обновить frontmatter.

## Пример
> User: "Запиши: нужно добавить тёмную тему в приложение"
> Agent: 
> 1. Title: "Тёмная тема"
> 2. Slug: "dark_theme"
> 3. Path: `workspace/inbox/2026-04-13_dark_theme.md`
> 4. Response: "✅ Записал. Добавить теги #ui #feature?"
